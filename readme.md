## Como rodar ?
primeiro compile o código utilizando o comando abaixo
```
gcc main.c ler.c motor.c -lm
```
depois execute o comando ``./a.out``.

## sobres os dados

 *  Rotação no melhor ponto: `rpm`
 *  Torque Máximo: `kgfm`
